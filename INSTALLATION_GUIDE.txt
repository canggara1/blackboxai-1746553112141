POS Application Installation Guide for Beginners
===============================================

This guide will help you set up and run the POS (Point of Sale) application with Midtrans QRIS payment integration.

Prerequisites
-------------
1. Node.js and npm installed (https://nodejs.org/)
2. MongoDB installed and running (https://www.mongodb.com/)
3. Internet connection to download dependencies and access Midtrans API

Step 1: Clone or Download the Project
-------------------------------------
- Download the project files or clone the repository to your local machine.

Step 2: Backend Setup
---------------------
1. Open a terminal and navigate to the backend directory:
   ```
   cd backend
   ```
2. Install backend dependencies:
   ```
   npm install
   ```
3. Create a `.env` file in the backend directory with the following environment variables:
   ```
   PORT=5000
   DB_URI=mongodb://localhost:27017/posapp
   MIDTRANS_CLIENT_ID=your_midtrans_client_id
   MIDTRANS_CLIENT_SECRET=your_midtrans_client_secret
   MIDTRANS_PARTNER_ID=your_midtrans_partner_id
   MIDTRANS_MERCHANT_ID=your_midtrans_merchant_id
   MIDTRANS_API_BASE_URL=https://merchants.sandbox.midtrans.com
   MIDTRANS_CHANNEL_ID=12345
   ```
   Replace the `your_midtrans_*` values with your actual Midtrans credentials.

4. Start the backend server:
   ```
   node server.js
   ```
   You should see a message: "Server running on port 5000" and "Connected to MongoDB".

Step 3: Frontend Setup
----------------------
1. Open a new terminal and navigate to the frontend directory:
   ```
   cd frontend
   ```
2. Install frontend dependencies:
   ```
   npm install
   ```
3. Start the frontend development server:
   ```
   npm start
   ```
4. Open your browser and go to `http://localhost:3000` to access the POS application.

Step 4: Using the POS Application
---------------------------------
- Manage inventory categories, variants, menu items, and discount items.
- Use the tab menu to add items to the shopping cart.
- Proceed to payment using either cash or QRIS GoPay.
- For QRIS payment, a QR code will be generated via Midtrans.
- Monitor payment status and view the success screen with receipt options.

Troubleshooting
---------------
- Ensure MongoDB is running before starting the backend.
- Verify all Midtrans credentials in the `.env` file are correct.
- Check terminal logs for any errors during startup.
- For API errors, check network requests in browser developer tools.
